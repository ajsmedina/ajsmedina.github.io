PokeBOX

Box

PokeBOX (codenamed BOX) is a project that was used to learn and explore PHP and MySQL while creating something functionally useful.

[PokeBOX] [GitHub]

Users can login using their trainer name to access their box, and then add Pokemon, remove Pokemon, or modify the Pokemon in their box. Then the user can create a team which gets analyzed for strengths and weaknesses.

PokeAPI

PokeAPI is an API that contains all the data from the Pokemon games in an easily-retrievable format.

PokeBOX uses PokeAPI to gather relevant data to be used in analyzing the teams, such as Pokemon stats, move types, and type matchups.

PHP code is run that retrieves the data from PokeAPI and stores it in a database using MySQL, allowing it to be used later on.

Unity and C#

This project was developed in the game engine Unity. Unity's physics engine and assets system allows for a simplified workflow that lets me focus on developing the game.

The code for Alien Entrepreneurs in the Wild Wild West is written in C# and implements many of Unity's build-in libraries to create functionality for the game.

The Full Package

Because this is a one-man project, everything is created by me. I programmed the game functionality, designed the levels, and drew the graphics for the game.

There are many parts that are still placeholders (such as the actual alien designs) since this game is still in development. However, 6 levels are currently playable for some fun team-building adventures.
